ULTIMATE SCRIPT

Gracias por ver nuestro Script, con él podremos navegar por un menú en el que podrás seleccionar 3 opciones:

  1. Crear un Disco Virtual.
  2. Borrar un Disco Virtual.
  3. Encriptar un Disco Virtual.

REQUISITOS ANTES DE USAR:

Es necesario ejecutar la consola de comandos de Powershell o Powershell ISE como Administrador, de lo contrario aparecerá un error.

Es necesario detener en Servicios locales, Detección de Hardware shell para poder crear un Disco Virtual

Es necesario Activar Hyper-V para crear y borrar Discos Virtuales.

Es necesario cambiar la politica de scripts ajenos al sistema, es necesario introducir el siguiente comando como ADMINISTRADOR.

Set-ExecutionPolicy Bypass

Recordar volver a cambiarlo a Default al terminar, para que no ocurra ningún percance con otros Scripts.

Todo el script esta desarrollado desde Powershell ISE proporcionada por Windows. Se permite el uso y compartir, pero no su distribución en masa ajena a este repositorio.
